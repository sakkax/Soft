```
# Installation
npm install -g truffle
truffle version
npm install -g ganache
```
Create a project
```
mkdir MetaCoin
cd MetaCoin
truffle unbox metacoin
truffle init
truffle compile
```
Test your truffle
```
 truffle test
truffle test ./path/to/test/file.js
```
